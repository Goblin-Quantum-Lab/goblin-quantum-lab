# SS-001: GitHub and Technical Workflow Basics

## Overview

This standalone helper course introduces GitHub workflows used by Goblin Quantum Lab learners and contributors.

## Helpful For

- Opening public-safe learner issues
- Reading repository files
- Understanding commits, branches, and pull requests
- Asking reproducible technical questions

## Relationship To Pathway Courses

This course is not a formal prerequisite by default. It may be recommended when learners need more confidence using GitHub during labs or public check-ins.
