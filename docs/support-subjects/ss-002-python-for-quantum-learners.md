# SS-002: Python for Quantum Learners

## Overview

This standalone helper course introduces the Python basics most useful for beginner quantum computing labs.

Tool and platform names are used for identification and compatibility only. No endorsement or affiliation is implied.

## Helpful For

- Running simple scripts and notebooks
- Reading variables, lists, and functions
- Installing packages in a virtual environment
- Understanding beginner simulator examples

## Relationship To Pathway Courses

This course is not a formal prerequisite by default. It may be recommended before or during lab-heavy courses.
