# Setup FAQ

## Do I Need Python?

QF-101 enrolled work expects local beginner tooling. Learners should be ready to use Python 3.11 or newer.

Start with [Local Python Environment](/docs/setup/local-python-environment).

## Do I Need Qiskit?

Not for public browsing. If an enrolled activity uses Qiskit, follow the enrolled course setup instructions.

## Do I Need IBM Quantum?

No. QF-101 does not require IBM Quantum, hardware access, cloud execution, API keys, access tokens, or paid services.

## Do I Need AWS Or Amazon Braket?

No. Do not create AWS resources for QF-101. Do not use Amazon Braket managed notebooks or paid QPUs for QF-101.

## Will This Cost Money?

No. QF-101 is designed for no-cost local learning after enrollment. See [No-Cost Learning Rules](/docs/setup/no-cost-learning-rules).

## What If `pip` Is Not Found?

Run pip through Python:

```bash
python -m pip --version
```

## What If `python` Is Not Recognized?

Try:

```bash
python3 --version
```

If that does not work, install Python 3.11 or newer and open a new terminal after installation. See [Troubleshooting](/docs/setup/troubleshooting).
