# Cost Safety FAQ

## Will QF-101 Cost Money?

No. QF-101 requires no paid services.

## Do I Need To Enter Credit-Card Details?

No. Do not enter credit-card details for QF-101.

## Do I Need AWS Or Amazon Braket?

No. Do not create AWS resources for QF-101. Do not use Amazon Braket managed notebooks or paid QPUs for QF-101.

## Do I Need IBM Quantum?

No. IBM Quantum hardware access may have free limited access, but QF-101 does not require hardware. Do not use IBM Pay-As-You-Go for QF-101.

## Can I Use Cloud Or Hardware Later?

Cloud or hardware execution is reserved for later advanced optional labs with explicit warnings. QF-101 uses local simulation only.

## What Should I Avoid Posting Publicly?

Never paste API keys, tokens, secrets, AWS access key values, IBM token values, private account details, private file paths, protected assessment materials, or protected assessment materials into public issues or submissions.

If a setup step asks for payment details or account credentials while you are doing QF-101, stop and use [Lab help](https://github.com/Goblin-Quantum-Lab/goblin-quantum-lab/issues/new?template=lab_help.yml).
