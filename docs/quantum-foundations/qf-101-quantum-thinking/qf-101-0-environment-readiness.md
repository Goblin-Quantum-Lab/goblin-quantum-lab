# QF-101.0: Environment Readiness and Simulation Setup

> **Enrollment Required**

QF-101 begins with QF-101.0: Environment Readiness and Simulation Setup. This unit prepares approved learners to set up a safe local environment before any QF-101 simulation labs.

## Public Preview

This public page is an overview only. The full interactive unit is not stored in the public repository and is not delivered through the public website.

Courses are enrolment-bound. Full course materials, labs, assignments, tests, and assessed work are released only through the enrolled learner process.

## What QF-101.0 Prepares Learners To Do

- Check local environment readiness.
- Understand the no-cost setup boundary for QF-101.
- Use local simulation before moving into beginner labs.
- Avoid posting secrets, account details, logs, or screenshots that expose private information.
- Ask for help through the public-safe learner support route when setup fails.

## Cost And Account Boundary

QF-101.0 is free but enrolment-bound. It does not require:

- a credit card
- cloud quantum hardware
- IBM Quantum Pay-As-You-Go
- AWS Braket
- Azure Quantum
- API keys
- access tokens
- paid services

## Access

Approved learners receive released coursework through the enrolled learner process. Access is manually reviewed, not instant, and bound to the learner's own GitHub account.

Support Subjects remain open to everyone. Full main pathway coursework is not stored publicly.
