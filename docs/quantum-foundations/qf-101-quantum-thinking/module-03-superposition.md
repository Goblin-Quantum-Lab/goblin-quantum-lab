# QF-101.3 Preview: Superposition Without Hype

This public page is a short preview only. Full lesson materials are released through the enrolled learner process.

## Public Preview

This module introduces superposition with careful beginner language. It avoids the misleading shortcut that a qubit is simply "both values at once" and instead frames superposition through state, possible outcomes, and measurement probabilities.

## Enrolled Learners Explore

- state before measurement
- possible outcomes
- probability distributions
- beginner intuition for simple transformations
- careful language around superposition
