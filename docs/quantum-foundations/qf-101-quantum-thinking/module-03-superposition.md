# Module 3: Superposition

## Purpose

This module introduces superposition with careful beginner language. Superposition is one of the first quantum ideas learners encounter, and it is often explained too loosely.

## State Before Measurement

Before measurement, a qubit can be in a state that is not simply the classical value `0` or the classical value `1`.

The state before measurement is used to predict possible measurement outcomes and their probabilities. The measurement still produces one classical outcome each time.

## Possible Outcomes

In a standard one-qubit measurement, the possible outcomes are:

- `0`
- `1`

Superposition does not mean that one measurement produces both outcomes. It means the pre-measurement state can involve both possible outcomes in the probability structure of the experiment.

## Probability Distribution

A superposition can produce a probability distribution over measurement outcomes. For example, a beginner experiment may be designed so that `0` and `1` are both plausible outcomes.

The observed counts from repeated shots help learners estimate that distribution.

## Hadamard Intuition

The Hadamard gate is often used in beginner examples because it can transform a definite starting state into a state that, when measured, tends to produce a balanced mix of `0` and `1` over many shots.

This is intuition, not a complete mathematical treatment. Later courses can introduce state vectors, amplitudes, phases, and the mathematics of quantum gates.

## Why "Both 0 and 1" Is Incomplete

Saying that a qubit is "both 0 and 1" can be a rough starting phrase, but by itself it is incomplete and often misleading.

Better beginner language:

> Before measurement, a qubit can be in a quantum state that assigns probabilities to possible measurement outcomes.

This language makes room for probability, measurement, and careful interpretation.

## Mini Task

Write a beginner-friendly explanation of superposition without saying only that a qubit is "both 0 and 1."

Aim for three to six sentences.

## Vocabulary

- **Superposition**: A quantum state that can involve multiple possible measurement outcomes.
- **Hadamard gate**: A common beginner gate used to create a balanced measurement pattern from a simple starting state.
- **Pre-measurement state**: The quantum state before a measurement is performed.
