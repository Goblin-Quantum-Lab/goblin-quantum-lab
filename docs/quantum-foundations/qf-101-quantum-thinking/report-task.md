# QF-101 Report Task Preview

This public page is a report-task overview only. Full report prompts, submission instructions, rubrics, protected review material, and assessed work are released only through the enrolled learner process.

## Public Preview

The enrolled report task asks learners to write a careful beginner observation report based on simulator-based experiment practice. The emphasis is on objective, expected result, method, observed result, interpretation, limitation, and conclusion.

## Access Boundary

The public repository does not publish completed report examples, protected assessment materials, protected grading materials, protected review material, or badge decision records.
