# Report Examples

This public page describes report-example categories at a high level. It does not publish completed course reports, protected assessment materials, protected lab results, assessment submissions, reviewer notes, or protected grading materials.

## Example Categories

- lab report structure
- algorithm implementation report structure
- literature review structure
- reproducibility note structure

## Public Boundary

Enrolled course-specific examples and reviewed submissions are released only through the enrolled learner process when appropriate. Do not publish completed enrolled-course reports as public examples.

For general writing guidance, visit [Report Writing](/docs/report-writing).
