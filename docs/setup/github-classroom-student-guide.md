# GitHub Classroom Student Guide

Goblin Quantum Lab uses GitHub-native public enrollment and support workflows. If an enrolled course activity uses GitHub Classroom or a similar no-cost workflow later, maintainers will provide instructions through the enrolled learner process.

QF-101 public overview pages do not require GitHub Classroom.

## Public Safety

- Use your own GitHub account.
- Read [GitHub Account Access](/docs/enrollment/github-account-access).
- Do not post sensitive personal data, passwords, API keys, access tokens, protected assessment materials, assignments, tests, reviewer notes, or badge decision records.
- Keep public help requests limited to public-safe setup or access questions.

## Public Boundary

This public page does not publish enrolled assignment links, lab files, tests, submission instructions, learner-specific review instructions, or protected workflow details.
