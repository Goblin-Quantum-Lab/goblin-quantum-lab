# Qiskit Local Simulator Notes

QF-101 does not require IBM Quantum, hardware access, cloud execution, API keys, access tokens, or paid services.

If an enrolled course activity uses Qiskit, follow the enrolled setup instructions and keep the work local unless the course explicitly says otherwise.

## Local-Only Rule For QF-101

- Use local simulation only.
- Do not connect to cloud backends for QF-101.
- Do not enter API keys, access tokens, secrets, or private account details.
- Do not use paid hardware, paid cloud notebooks, or managed quantum services for QF-101.

## Public Boundary

This public page does not publish enrolled QF-101 lab files, assignment prompts, tests, or submission instructions.
