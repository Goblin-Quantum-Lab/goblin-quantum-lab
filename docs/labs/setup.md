# Lab Setup Overview

This public page gives setup readiness guidance only. It does not publish enrolled lab files, lab instructions, assignments, tests, or submission instructions.

QF-101 is designed for no-cost local simulator work after enrollment. It does not require cloud quantum hardware, paid services, credit cards, API keys, or access tokens.

## Public Safety

Never paste API keys, access tokens, secrets, private account details, official solutions, protected assessment materials, enrolled-only instructions, or reviewer notes into public issues, reports, or code files.

Ask for public-safe help with the [Lab help issue form](https://github.com/Goblin-Quantum-Lab/goblin-quantum-lab/issues/new?template=lab_help.yml).
