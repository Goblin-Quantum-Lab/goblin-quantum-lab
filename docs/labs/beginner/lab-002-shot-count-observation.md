# Lab 002: Shot Count Observation

## Purpose

This lab shows why repeated shots matter. You will compare result patterns from 10 shots, 100 shots, and 1,000 shots.

The goal is to observe how sample size affects the pattern of counts.

## Learning Goals

By the end of this lab, you should be able to:

- Explain what a shot is.
- Compare count tables from different shot counts.
- Describe why small samples may appear unstable.
- Explain why larger shot counts usually provide a clearer distribution.
- Avoid overstating what one small sample proves.

## Suggested Setup

Use a beginner one-qubit experiment that can be repeated with different shot counts. A common setup is a simple circuit that produces more than one possible measurement outcome.

If you are not running code yet, use public-safe result tables from a tutorial, instructor example, or future Goblin Quantum Lab lab file.

Do not use private credentials, paid services, or API keys for this beginner lab.

## Learner Task

Run or inspect the same experiment using:

- 10 shots
- 100 shots
- 1,000 shots

For each shot count:

1. Record the observed counts.
2. Describe the pattern in plain language.
3. Compare the pattern with the other shot counts.
4. Explain why small samples may appear unstable.
5. Explain why larger shot counts usually provide a clearer distribution.
