# Lab 001: Single-Qubit Measurement

## Purpose

This lab introduces a simple prepared quantum state and its measurement result.

You may inspect a provided example, run a simulator example, or use a classroom notebook if one is available. The important work is to record what you expected, what you observed, and what the result can and cannot show.

## Learning Goals

By the end of this lab, you should be able to:

- Describe a simple one-qubit measurement experiment.
- Record an expected result before inspecting the final counts.
- Record an observed result after measurement.
- Compare expectation and observation.
- State one limitation of the experiment.

## Suggested Setup

Use any public-safe beginner quantum simulator or notebook environment. If you are not running code yet, inspect a simple one-qubit measurement example provided by an instructor, tutorial, or future Goblin Quantum Lab lab file.

Do not use private credentials, paid services, or API keys for this beginner lab.

## Experiment Sketch

Use a one-qubit circuit or example that:

1. Prepares a simple one-qubit state.
2. Measures the qubit in the computational basis.
3. Reports counts for outcomes such as `0` and `1`.

## Learner Task

Complete the following:

1. Record the prepared state or circuit you inspected.
2. Record the number of shots, if available.
3. Write the expected result before inspecting the final counts.
4. Record the observed result.
5. Explain whether the observed result matches your expectation.
6. State one limitation of the experiment.
