# GitHub Account Access

Goblin Quantum Lab is a free public static site. Anyone can read the public lessons, labs, templates, and learning guidance without creating a site account.

Learners who want guided enrollment, public submissions, lab help, check-ins, badge review, or assessed learning milestones need a GitHub account. For now, GitHub is the learner identity used by the public workflow.

## How Access Works

- Public reading access is open to everyone.
- Learner enrollment and submissions use GitHub Issues in the public repository.
- Your GitHub username is used as the learner identity for public workflow activity.
- Assessed work, milestone requests, check-ins, and public submissions are account-bound to the GitHub account that posts them.
- Advanced work may be prerequisite-gated. Maintainers may ask learners to complete earlier public courses, labs, check-ins, or milestone tasks before moving into advanced material.
- Badge and completion review is manual and may take up to 30 business days.

Goblin Quantum Lab does not provide a custom login system, site account, paid learning portal, or separate learner database for the public site.

## Public Safety

GitHub Issues are public. Do not submit sensitive personal data, private contact details, school IDs, home addresses, secrets, tokens, API keys, official solutions, or protected assessment materials.

If something should not be visible to the public, do not post it in a GitHub Issue.

## Review Expectations

Enrollment, course access guidance, badge requests, and completion review are handled manually by maintainers or approved reviewers. Review may take up to 30 business days. Submitting a request does not guarantee approval or immediate advancement.

Advanced learning areas may require visible public progress from earlier work. This keeps support manageable and helps learners build a clear record of study, practice, and reflection.
